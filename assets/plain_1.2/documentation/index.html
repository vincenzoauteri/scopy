HTML Files and Structure

About this theme

Plain is a responsible HTML template with 12-column grid based on popular Twitter Bootstrap framework. This theme is suitable as a clean and easily maintainable template for any business, portfolio or single-project site, with support for responsive layouts, latest HTML5 & CSS3 techniques and many other user interface components and javascript plugins, which are described in this documentation. 

Basic structure

All the pages have been divided into four important sections #header, #content, #footer and #copyright as show on this simplified HTML code:

<!-- Header -->

<header id="header">
  <nav class="navbar">
    { NAVIGATION CONTENT }
  </nav>
</header>

<!-- Content -->

<section id="content" class="container">
  { CONTENT }
</section>

<!-- Footer -->

<footer id="footer">
  <div class="container">
    { FOOTER CONTENT }
  </div>
</footer>

<!-- Copyright -->

<footer id="copyright">
  <div class="container">
    { COPYRIGHT CONTENT }
  </div>
</footer>

Columns

Template is divided into rows and columns, with respective classes .row and .span*, where rows divide horizontal parts of the page and columns divide each row into smaller vertical sections. Since this template is based on 12-column grid system, you can divide each row into various number of vertical sections as long as they sum up at twelve (12). Look at the examples below for better explanation and some real examples:


Full-width: To achieve a full-width column, use the code below:

<div class="row">
  <div class="span12">
    { CONTENT }
  </div>
</div>


Two equal sections: To split the page into two sections with the same width (1/2 + 1/2 layout)

<div class="row">
  <div class="span6">
    { CONTENT #1 }
  </div>
  <div class="span6">
    { CONTENT #2 }
  </div>
</div>

Notice that .span6 and .span6 makes up to 12. Remeber, you can combine any number of .span* classes as long as they sum up as 12. 


Two sections with different width (2/3 + 1/3 layout)

<div class="row">
  <div class="span8">
    { CONTENT #1 }
  </div>
  <div class="span4">
    { CONTENT #2 }
  </div>
</div>


Three equal sections (1/3 + 1/3 + 1/3 layout)

<div class="row">
  <div class="span4">
    { CONTENT #1 }
  </div>
  <div class="span4">
    { CONTENT #2 }
  </div>
  <div class="span4">
    { CONTENT #3 }
  </div>
</div>

Container

There's one more special class which is being used across all HTML files, which is .container. This class determines the maximum width of a page and also makes sure that page is center-aligned. It also determines rendering of a template on mobile devices and tablets.

Header and navigation

 

Example header and navigation structure:

<header id="header">
  <nav class="navbar">
    <div class="navbar-inner">
      <div class="container">

        <!-- Logo -->
        <a class="brand" href="home.html">
          Plain { PLACE YOUR LOGO HERE }
        </a>

        <!-- Navigation -->
        <ul class="nav">

          <!-- Example of item -->
          <li><a href="home.html">Home</a></li>

          <!-- Example of selected item - notice the .active class -->
          <li class="active"><a href="work.html">Our work</a></li>

          <!-- Example of 2nd level navigation - notice the .dropdown class and <ul> element -->
          <li class="dropdown">
            <a href="icons.html">Features</a>
            <ul>
              <li><a href="{URL}" title="">{TITLE}</a></li>
              ...
            </ul>
          </li>
          ...

        </ul>
      </div><!-- end .container -->
    </div><!-- end .navbar-inner -->
  </nav>
</header>

How to change or modify logo?
Place your logo inside the a.brand container. To modify it's appearance, find and modify respective .brand class in css/style.css stylesheet file.

How to make navigation item selected?
To make navigation item active (selected), add .active class to it's <li> parent container. 

How to create dropdown (2nd level) navigation?
To crete dropdown navigation item, add .dropdown class to it's parent <li> element and place <ul></ul> element inside of it.

Homepage slider

Slider which you can see at the homepage comes in two different versions. Based on a class of #carousel element, you can choose between default slider or slider with macbook placeholder. For Apple Macbook version, just add .macbook class to #carousel element as show on code below:

<div id="carousel" class="carousel slide macbook" rel="carousel">

  <! -- Macbook image -->
  <img src="img/carousel-macbook.png" class="mac" alt="" />

  <!-- Carousel items -->
    <div class="carousel-inner">
    <!-- Item -->
      <div class="item active">
        <img src="{SRC}" alt="" />
      </div>
    <!-- end Item -->
    ... { any number of items here } ...
    <!-- Item -->
    <div class="item">
      <img src="{SRC}" alt="" />
    </div>
    <!-- end Item -->
  </div>

  <!-- Carousel navigation -->
  <a class="carousel-control left"  href="#carousel" data-slide="prev">&lsaquo;</a>
  <a class="carousel-control right" href="#carousel" data-slide="next">&rsaquo;</a>

</div>

Subpages - Our work

Projects are placed in #portfolio-items and every project is a single element of <article> type. Every project uses data-tags attribute to determine project category when projects are being filtered by #portfolio-filter controls. Tags inside data-tags attribute should be separated by comma. To avoid any possible problems, only lowercase letters without spaces should be used to describe project tags. 
 
Example of one project item:
<article class="span4 project" data-tags="web">
  <div class="thumbnail">
    <a href="project.html" title="">
      <img alt="" src="{IMAGE SRC}" />
    </a>
    <div class="project-description">
      <b class="name">{PROJECT NAME}</b>
      <i class="icon-thumbs-up"></i>
    </div>
  </div>
</article>
Example of filter:
<ul id="portfolio-filter" class="nav nav-tabs">
  <li class="active"><a href="#" class="filter" data-filter="*">All projects</a></li>
  <li><a href="#" class="filter" data-filter=".web">Projects from "web" category</a></li>
</ul>

CSS Files and Structure

This theme comes with 5 CSS files:

  1. css/bootstrap/bootstrap.min.js
  2. css/bootstrap/bootstrap-responsive.min.js
  3. css/bootstrap-override.css
  4. css/style.uncompressed.css
  5. css/font-awesome/font-awesome.css (+ css/font-awesome/font-awesome-ie7.css for IE compatibility)

Please note that the first three files are being loaded with css @import rule at the top of css/style.uncompressed.css stylesheet

Little more about each file:

css/bootstrap/bootstrap.min.js
Twitter bootstrap framework. A base twitter bootstrap CSS files, which defines various elements like  columns, navigation, tabs, icons, forms and buttons.  

css/bootstrap/bootstrap-responsive.min.js
Twitter bootstrap support for responsive layou, so the template can be displayed on mobile devices or tablets.

css/bootstrap-override.css
This file is not an official part of twitter bootstrap, it contains few overrides of media queries of original theme. The most important change (while keeping twitter bootstrap's responsive functionality) is that it restricts .container class of this template to be at most 1024px wide.

css/font-awesome/font-awesome.css (+ font-files)
This file includes custom font which is being used for all icons which you can see in this template. More information about Font Awesome 100% scalable icon pack can be found at http://fortawesome.github.com/Font-Awesome/. 

css/style.min.css - Minified version
css/style.original.css - Original version

This is the most important style of our template. To make any changes to this template, you should only need to modify this file. The file is divided info different sections, while every section take name from a subpage of template features.

JavaScript plugins and scripts

This template comes with 6 Javascript files:

  1. jquery-1.7.2.min
    jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  2. bootstrap.min
    Twitter bootstrap base JavaScript file, which allows us to use many javascript components like tabs, accordion menus, dropdowns, tooltips and popovers, alert messages and so on. More information and examples can be found at http://twitter.github.com/bootstrap/javascript.html
  3. jquery.isotope.min.js
    Used to filter portfolio items
  4. jquery.hotkeys
    Keyboard shortcuts functionality (e.g. navigating to previous/next project by pressing Left/Right arrow)
  5. jflickrfeed.js 
    Flickr API functions for flickr widget

  6. functions.js
    The most important file for our theme, which defines some basic javascript functions. 

    This file is divided into different sections, while every section is responsible for different functionality. 

    Twitter bootstrap components - carousel, tooltip, popover 
    Initialization of some twitter bootstrap javascript plugins

    Flickr feed

    Zoom and link overlays
    Zoom and link functionality of out thumbnails (+ and link icons which appear after you move your mouse over thumbnail - e.g. portfolio item)

    Responsible navigation
    Creates a collapsible top navigation for mobile devices

    Portfolio 
    Basic functionality for filtering portfolio items. 

    Back to top button

    Contact form
    Provides basic functionality for contact form (sending emails through ajax and included .php script)

    Newsletter subscription form
    Provides functionality for newsletter subscription form (works through ajax and included .php script)

    Keyboard shortcuts
    Initialization of keyboard shortcuts. You can easily define your own shortcuts by using this code:

    <a rel="shortcut" data-key="k" href="http://google.com">Press "k" to visit google.com</a>
    This code will result in navigation your visitor to google.com after pressing "k" key. 

PHP Scripts

Plain HTML template comes with two php scripts and a configuration file. These scripts provide server-side support for sending emails for contact and newsletter form.

config.php

Location: php/config.php

Configuration file for all other scripts. It contains option for setting up your email address for contact form and newsletter form. Please open this file and replace $your_email with your own email address.

contact-form.php

Location: php/contact-form.php

Setting up the contact form is a simple task. All you need is to open php/config.php file and replace $your_email value with your own email address.
This script takes care of the rest - it accepts arax reqests made from contact-form page (contact.html), do some basic validation and notifies you about new messages.

newsletter-form.php

Location: php/newsletter-form.php

This script notifies you about newsletter subscriptions. You should set up your email address in php/config.php before using it.

Important notice

This theme comes with support for font-awesome web-font, although the main font files can't be included in template package.

To ensure compatibility with this free webfont, please follow these steps:

1. Download font-awesome from http://fortawesome.github.com/Font-Awesome/
2. Copy content from "font" directory to css/font-awesome/font/ and ensure that these files are present in your template as follows:

/css/font-awesome/font/fontawesome-webfont.eot
/css/font-awesome/font/fontawesome-webfont.svg
/css/font-awesome/font/fontawesome-webfont.ttf
/css/font-awesome/font/fontawesome-webfont.woff
