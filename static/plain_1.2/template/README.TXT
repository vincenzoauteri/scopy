**************************************************************

	This theme comes with support for font-awesome web-font, although the main font files can't be included in template package. 
	To ensure compatibility with this free webfont, please follow these steps:

	1. Download font-awesome from http://fortawesome.github.com/Font-Awesome/ 
	2. Copy content from "font" directory to css/font-awesome/font/ and ensure that these files are present in your template as follows:
		css/font-awesome/font/fontawesome-webfont.eot
		css/font-awesome/font/fontawesome-webfont.svg
		css/font-awesome/font/fontawesome-webfont.ttf
		css/font-awesome/font/fontawesome-webfont.woff
